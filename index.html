博客简介

这并不仅仅是一个技术博客，还是一个记录生活的博客。

这个博客域名购买于百度云，为什么选择百度云，因为便宜呀。由Hexo强力驱动，托管在Github上，主题为 NexT.Gemini。

这篇文章主要记录了博客的搭建过程、NexT主题的嵌入以及域名的绑定。关于以后博客的一些后续优化，将会另开篇幅。

博客搭建过程

  • 相关环境准备

    • 安装Git。请戳此处直接下载，安装直接一路next直到结束。

    • 安装Node.js。请戳此处直接下载，同理安装一路直接next直到结束。

    • 配置Github。

      • 在www.github.com上注册一个账号，然后Crew a new reposiroty（用小号演示）

      • 注意你的Repository name格式为“yourname.github.io”，yourname为你的github用户名。

  • 生成SSH密钥

    • 安装好Git后，在桌面空白部分右键点击Git Brash Here
    • 然后输入命令 ssh-keygen -t rsa -C “your mail adress name”，然后enter,输密码，密码是不可见的。
    • 在C:\Users\Administrator.ssh下，得到两个文件id_rsa和id_rsa.pub。

  • 在GitHub上添加SSH密钥

    • 打开刚才得到的两个文件中的id_rsa.pub，把里面的内容全部复制出来
    • 登陆Github->settings->SSH and GPG keys
    • 点击New SSH Key，Title随便写（最好写id_rsa.pub)，Key的内容粘贴刚才复制的内容，然后OK。
    • 测试SSH证书是否正确， ssh -T git@github.com，如果出现下列文字表示到目前为止你的操作正确。”You’ve successfully authenticated,but..”

​

  • 安装Hexo以及Hexo的相关配置

    • 在D盘新建一个空白文件夹，进入空白右键Git Brash Here,输入node，如果出来一个右箭头，说明Node.js安装成功了，然后输入Ctrl+C退出。

    • 输入 npm install -g hexo-cli 安装Hexo，这时候可能出现错误，具体错误百度。

    • 输入 hexo version，如果出现下图，说明Hexo安装成功。

    • 安装成功后，打开这个文件夹中的 _config.yml，在里面的 deploy字段添加自己的信息，注意冒号后面有一个空格

    • 输入 hexo s –debug，如果没有报错的话，在本地浏览器中输入 http://localhot:4000，即可看见本地的网站，然后按Ctrl +C 结束预览

    • 输入 hexo g ,生成网页

    • 输入 hexo d,然后登陆 yourname.github.io就可以看见你的博客了
