JAVA的几个常用类总结

发表于 2017-10-17

Java有一系列功能强大的可重用类，分别在不同的包中，这些包按照功能可以划分为：

  1. 语言包 java.lang (language)，主要含有与语言相关的类，由解释程序自动加载，不需要显示说明。
  2. 输入输出包 java.io (input/output)，提供了对不同的输入和输出的支持。
  3. 实用程序包 java.util (utility)，包括许多具有特功能的类，比如日期、向量、哈希表等。
  4. 小程序应用包 java.applet，用来构建包含于HTML页内的applet必不可少的类。
  5. 图形用户接口包 java.swing & java.awt，提供了创建图形用户界面的类。
  6. 网络包 java.net，含有与网络操作相关的类，如TCP sockets、URL等。

其中，前三种包被称为java的基础包，本文将从以下两个包分别分析其中包含的类。

java.lang

  • Object类。是除自己之外，所有java类的父类，但是接口不继承该类。这个类包含以下几个函数。

    • protected Object clone() 产生一个相同的类并且返回给调用者，使用这个函数的类必须要实现java.lang.Cloneable接口

    • boolean equals(Object obj) 检查一个对象与调用这个函数的对象是否相等

    • protected void finalize() 终止，释放系统资源

    • Class< >getClass 得到对象的类

    • int hashCode() 返回给调用者此对象的哈希码，通常用于Hash集合

    • void wait(…) 等待线程

    • void notifyall() 唤醒线程

    • String toString() 根据调用对象返回对象的字符串形式

      关于这几个函数的应用以及深入研究请访问此链接。

  • 基本数据类型的类：String、Boolean、Character、Byte、Interger、Short、Long、Float、Double。这些都是基本类型的包装类。

    • 说明一下String、StringBuffer、StringBuilder的区别：请访问此链接。

  • Math类，封装了一些数学函数和常量。（三角、反三角、乘方、舍入..）

  • System类，有标准输入、标准输出和错误输出流。其不能被实例化，但是

    • 有三个非常有用的对象：

      • static PrintStream err “错误输出流”
      • static InputStream in “标准输入流”
      • static PrintStream out “标准输出流”

    • 常用的方法：

      • static void arraycopy(..) 从指定数组源复制一个数组
      • static long currentTimeMills() 返回以毫秒为单位的当前时间
      • static void exit(int status) 终止当前正在运行的JAVA虚拟机
      • static void gc() 运行垃圾回收器

  • Runtime类，封装了运行时的环境，每个Java应用程序都有一个Runtime类实例。需要用的话请访问此链接。

  • Thread类，包含线程相关的一系列函数。

  • Throwable，异常类。

java.util

  • 日期时间类，常用的方法有：
    • public int getYear() 、getMonth() 、getDate()、、、
    • public boolean before(Date when) 、after 、equal
    • toString()
  • Random，用于生成伪随机数流
  • Vector，大致用法和C++一致
  • ArrayList，与Vector大致相同，都实现了List接口，但是Vector是同步的，ArrayList是不同步的。
  • HashSet，大致用法与C++一致，set接口，由Hash表支持。
  • Iterator，迭代器，常用的几个用法：
    • boolean hasNext() 如果仍有元素可以迭代，则返回true
    • E next() 返回迭代的下一个元素
    • void remove() 从迭代器指向的集合中移除迭代器返回的最后一个元素（可选操作）

关于我的博客

发表于 2017-10-16

博客简介

这并不仅仅是一个技术博客，还是一个记录生活的博客。

这个博客域名购买于百度云，为什么选择百度云，因为便宜呀。由Hexo强力驱动，托管在Github上，主题为 NexT.Gemini。

这篇文章主要记录了博客的搭建过程、NexT主题的嵌入以及域名的绑定。关于以后博客的一些后续优化，将会另开篇幅。

博客搭建过程

  • 相关环境准备

    • 安装Git。请戳此处直接下载，安装直接一路next直到结束。

    • 安装Node.js。请戳此处直接下载，同理安装一路直接next直到结束。

    • 配置Github。

      • 在www.github.com上注册一个账号，然后Crew a new reposiroty（用小号演示）

        注意你的Repository name格式为“yourname.github.io”，yourname为你的github用户名。

  • 生成SSH密钥

    • 安装好Git后，在桌面空白部分右键点击Git Brash Here
    • 然后输入命令 ssh-keygen -t rsa -C “your mail adress name”，然后enter,输密码，密码是不可见的。
    • 在C:\Users\Administrator.ssh下，得到两个文件id_rsa和id_rsa.pub。

  • 在GitHub上添加SSH密钥

    • 打开刚才得到的两个文件中的id_rsa.pub，把里面的内容全部复制出来
    • 登陆Github->settings->SSH and GPG keys
    • 点击New SSH Key，Title随便写（最好写id_rsa.pub)，Key的内容粘贴刚才复制的内容，然后OK。
    • 测试SSH证书是否正确， ssh -T git@github.com，如果出现下列文字表示到目前为止你的操作正确。”You’ve successfully authenticated,but..” !

​

  • 安装Hexo以及Hexo的相关配置

    • 在D盘新建一个空白文件夹，进入空白右键Git Brash Here,输入node，如果出来一个右箭头，说明Node.js安装成功了，然后输入Ctrl+C退出。

    • 输入 npm install -g hexo-cli 安装Hexo，这时候可能出现错误，具体错误百度。

    • 输入 hexo version，如果出现下图，说明Hexo安装成功。

    • 安装成功后，打开这个文件夹中的 _config.yml，在里面的 deploy字段添加自己的信息，注意冒号后面有一个空格

    • 输入 hexo s –debug，如果没有报错的话，在本地浏览器中输入 http://localhot:4000，即可看见本地的网站，然后按Ctrl +C 结束预览

    • 输入 hexo g ,生成网页

    • 输入 hexo d,然后登陆 yourname.github.io就可以看见你的博客了

NexT主题嵌入

访问网站，里面有详细的NexT主题设置。

域名绑定

域名绑定要创建一个CNAME文件，里面内容只有你的域名，但是如果放在master分支下，每次提交项目的时候都会被重写，每次都要手动加入，所以将CNAME文件放在source文件夹中。

相关命令普及

ssh-keygen 为”ssh”生成、管理和转换认证密钥，支持rsa和dsa
npm install hexo -g 安装hexo
hexo init 初始化
hexo g g = generate ,生成
hexo d d = deploy ,部署
hexo s s = server ,启动服务预览
