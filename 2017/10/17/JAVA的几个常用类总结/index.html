JAVA的几个常用类总结

发表于 2017-10-17

Java有一系列功能强大的可重用类，分别在不同的包中，这些包按照功能可以划分为：

  1. 语言包 java.lang (language)，主要含有与语言相关的类，由解释程序自动加载，不需要显示说明。
  2. 输入输出包 java.io (input/output)，提供了对不同的输入和输出的支持。
  3. 实用程序包 java.util (utility)，包括许多具有特功能的类，比如日期、向量、哈希表等。
  4. 小程序应用包 java.applet，用来构建包含于HTML页内的applet必不可少的类。
  5. 图形用户接口包 java.swing & java.awt，提供了创建图形用户界面的类。
  6. 网络包 java.net，含有与网络操作相关的类，如TCP sockets、URL等。

其中，前三种包被称为java的基础包，本文将从以下两个包分别分析其中包含的类。

java.lang

  • Object类。是除自己之外，所有java类的父类，但是接口不继承该类。这个类包含以下几个函数。

    • protected Object clone() 产生一个相同的类并且返回给调用者，使用这个函数的类必须要实现java.lang.Cloneable接口

    • boolean equals(Object obj) 检查一个对象与调用这个函数的对象是否相等

    • protected void finalize() 终止，释放系统资源

    • Class< >getClass 得到对象的类

    • int hashCode() 返回给调用者此对象的哈希码，通常用于Hash集合

    • void wait(…) 等待线程

    • void notifyall() 唤醒线程

    • String toString() 根据调用对象返回对象的字符串形式

      关于这几个函数的应用以及深入研究请访问此链接。

  • 基本数据类型的类：String、Boolean、Character、Byte、Interger、Short、Long、Float、Double。这些都是基本类型的包装类。

    • 说明一下String、StringBuffer、StringBuilder的区别：请访问此链接。

  • Math类，封装了一些数学函数和常量。（三角、反三角、乘方、舍入..）

  • System类，有标准输入、标准输出和错误输出流。其不能被实例化，但是

    • 有三个非常有用的对象：

      • static PrintStream err “错误输出流”
      • static InputStream in “标准输入流”
      • static PrintStream out “标准输出流”

    • 常用的方法：

      • static void arraycopy(..) 从指定数组源复制一个数组
      • static long currentTimeMills() 返回以毫秒为单位的当前时间
      • static void exit(int status) 终止当前正在运行的JAVA虚拟机
      • static void gc() 运行垃圾回收器

  • Runtime类，封装了运行时的环境，每个Java应用程序都有一个Runtime类实例。需要用的话请访问此链接。

  • Thread类，包含线程相关的一系列函数。

  • Throwable，异常类。

java.util

  • 日期时间类，常用的方法有：
    • public int getYear() 、getMonth() 、getDate()、、、
    • public boolean before(Date when) 、after 、equal
    • toString()
  • Random，用于生成伪随机数流
  • Vector，大致用法和C++一致
  • ArrayList，与Vector大致相同，都实现了List接口，但是Vector是同步的，ArrayList是不同步的。
  • HashSet，大致用法与C++一致，set接口，由Hash表支持。
  • Iterator，迭代器，常用的几个用法：
    • boolean hasNext() 如果仍有元素可以迭代，则返回true
    • E next() 返回迭代的下一个元素
    • void remove() 从迭代器指向的集合中移除迭代器返回的最后一个元素（可选操作）
